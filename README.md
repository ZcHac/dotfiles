# dotfiles

## Introduction

This repo contains all the dotfiles for my personal Arch Linux ricing configuration, and possibly auto
installation script to enable fast bootstrapping from scratch!

Currently included utilities:

- Command Line Shell:
  - bash
  - zsh

- Terminal:
  - alacritty

- Multiplexer:
  - tmux

- Text editor:
  - nvim

- Repository:
  - git

- Font:
  - NerdFont
